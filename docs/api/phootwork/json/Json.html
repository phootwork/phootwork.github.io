static array decode(string $json, int $options = 0, int $depth = 512)

Decodes a JSON string to an array.

Parameters

string $json The json string being decoded. This only works with UTF-8 encoded strings.
int $options Bitmask of JSON decode options. Currently only OBJECT_AS_ARRAY, BIGINT_AS_STRING is supported (default is to cast large integers as floats)
int $depth User specified recursion depth.

Return Value

array Returns the value encoded in json in appropriate PHP type. Values true, false and null are returned as TRUE, FALSE and NULL respectively.

Exceptions

JsonException if something gone wrong
