Controls whether case-folding is enabled for this XML parser. Enabled by default.

Data Type: integer

Data Type: integer

Whether to skip values consisting of whitespace characters.

Data Type: integer
OPTION_TARGET_ENCODING

Sets which target encoding to use in this XML parser. By default, it is set to the same as the source encoding used by XmlParser::construct(). Supported target encodings are ISO-8859-1, US-ASCII and UTF-8.

Data Type: string
